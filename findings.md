# Findings

## 2026-03-19

- 用户给出的待修问题只有 4 项，来源文件位于桌面，内容明确，不需要额外需求澄清。
- 当前仓库已经存在一批未提交修改，主要集中在 AI/编辑器工具栏相关文件，后续修复需要避免覆盖这些改动。

- `文件版本历史` 的 UI、提交对话框、git log/show 基本都已存在，但仓库没有 `git init` 能力，非 git 工作区下历史几乎不可用。
- `@ 文件搜索` 的底层文件索引存在，但弹层只做了简单 `includes` 过滤并硬截断 20 条，没有评分排序，也没有对当前工作区上下文做任何提升。
- `LaTeX 中文路径` 还需要继续下钻 Rust 编译链路；目前已确认问题不在前端 compile 调用入口，而更可能在后端编译命令与路径处理细节。
