# Legacy Shoulders Reference

This directory contains historical documentation inherited from the upstream Shoulders codebase.

These files are kept for repository archaeology and selective implementation reference. They should **not** be treated as the default source of truth for current Altals behavior.

If you are maintaining current Altals, start from:

- [../_MAP.md](../_MAP.md)
- [../../README.md](../../README.md)

Use the files in this directory only when you specifically need historical context about removed hosted systems such as:

- hosted auth and account flows
- admin and analytics backends
- peer review and triage web products
- historical website/backend architecture
